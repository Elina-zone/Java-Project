# Inventory Management System in Java

This repository contains a console-based Inventory Management System implemented in Java. It provides a structured interface for admins and staff to manage the inventory, products, sales, customers, suppliers, and categories in an organized way.

# key Features:

User Authentication.
Admin and Staff login with separate credentials.
Role-based access to different functionalities.
Admin Functionalities:
Manage Categories: Add, View, Delete.
Manage Products: Add, View, Delete.
Manage Customers: Add, View, Delete.
Manage Sales Orders: Add, View, Delete.
Manage Suppliers: Add, View, Delete.
Logout and navigate back to Login Menu.
Staff Functionalities:
View Products.
View and Add Sales Orders.
View Customers.
Logout and navigate back to Login Menu.
Interactive Menu System:
User-friendly console menus for easy navigation.
Clear prompts for actions and input validation.

#Technologies Used:

Java: Core Java for console-based operations.
Scanner Class for user input handling.


This project is an educational example of how to implement a role-based inventory management system using Java. It demonstrates menu-driven programming, CRUD operations, and user authentication while providing a foundation for future enhancements, like integrating a database or GUI.
